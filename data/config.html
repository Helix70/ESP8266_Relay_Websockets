Configuration

Board Name

The board name will appear at the top of the main page and in the browser tab.

Startup Behavior

If enabled, the board will delay for the selected number of seconds on startup before beginning relay operations, up to 5 minutes.

When enabled, startup performs a Wi-Fi scan and attempts to connect to the strongest AP matching your configured SSID. If disabled, startup uses normal connect behavior.

Wi-Fi Status

Clearing credentials restarts into provisioning mode.

Network Configuration

If no static IP is specified, DHCP is used automatically. Network Configuration changes require reboot.
