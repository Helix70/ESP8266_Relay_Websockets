Relay Board

Board Name

The board name will appear at the top of the main page and in the browser tab.

Network Configuration

If no static IP is specified, DHCP is used automatically. Static IP changes require reboot.

Board Hardware

MCU type is set at firmware compile time. Changing the relay count requires a restart.

Relay Modes

If enabled, the board will delay for the selected number of seconds on startup before beginning relay operations, up to 5 minutes.

If enabled, relays can be configured to automatically turn off after a set timeout.

If enabled, certain relay groups will be mutually exclusive (only one can be on at a time).

If enabled, relays can be configured to pulse for a set duration.
